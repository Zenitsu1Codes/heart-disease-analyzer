# ❤️ Heart Disease Analyzer (Machine Learning Project)

> A machine learning project that predicts the likelihood of heart disease based on patient medical data.

---

## 📌 Overview

This project uses machine learning techniques to analyze clinical data and predict whether a patient is likely to have heart disease.

It follows a complete data science workflow:

* Problem definition
* Data analysis
* Feature engineering
* Model training
* Evaluation

The goal is to build a model with high accuracy (target: **95%**) to assist in early detection.

---

## 🎯 Problem Statement

> Given clinical parameters about a patient, can we predict whether they have heart disease?

---

## 📊 Dataset

* Source: UCI Machine Learning Repository (Cleveland dataset)
* Also available on Kaggle
* Contains medical attributes such as:

  * Age
  * Sex
  * Chest pain type
  * Blood pressure
  * Cholesterol levels
  * Heart rate

---

## ✨ Features

* ✅ Data preprocessing and cleaning
* ✅ Exploratory Data Analysis (EDA)
* ✅ Feature analysis
